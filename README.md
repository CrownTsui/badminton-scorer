# 🏸 羽毛球记分器

零安装、即开即用的羽毛球比赛记分网页。手机上单手操作，每得一分点一下即可。

## 功能

- **快速记分**：两侧大按钮，点即+1，300ms 防抖防误触
- **标准规则**：21 分制、20 平需领先 2 分、30 分上限、三局两胜
- **发球权**：自动切换，视觉标识清晰可见
- **撤销**：误触可撤销（当前局内），Ctrl+Z 快捷键
- **暂停/继续**：比赛控制，空格键快捷键
- **换场提示**：局间和第三局 11 分自动提醒
- **持久化**：LocalStorage 自动保存，刷新不丢分
- **无障碍**：屏幕阅读器支持，键盘导航，高对比度
- **横屏适配**：双手握持操作布局

## 快捷键

| 按键 | 操作 |
|------|------|
| A / ← | A 队得分 |
| B / → | B 队得分 |
| 空格 | 暂停/继续 |
| Ctrl+Z | 撤销 |

## 使用方式

### 直接打开
1. 下载 `index.html`
2. 用浏览器打开即可

### 部署到 GitHub Pages
```bash
# 在仓库设置中启用 GitHub Pages，选择 main 分支根目录
git push origin main
```

## 比赛规则

- 三局两胜，每局 21 分
- 20 平后需领先 2 分才获胜
- 29 平后先到 30 分者胜
- 第一局、第二局结束后交换场地
- 第三局任一方到 11 分时交换场地

## 技术

纯 HTML + CSS + JS，零依赖。约 600 行代码，所有逻辑内联。
